# Fiber Booking

## General

- room booking sample
- backend go fiber
- frontend nextjs
- graphql

## Run

### Backend

```bash
cd backend
go run main.go
# or watch
go run github.com/pilu/fresh
```

### Frontend

```bash
cd frontend
npm i
npm run dev
```
